# Plugin compatibility
This document describes compatibility of the versions of this plugin with the Racetrack platform.

| Plugin version | Compatible Racetrack version |
|----------------|------------------------------|
| 2.9.0          | `> 2.17.0`                   |
| 2.8.0          | `>= 2.15.0`                  |
| 2.6.2          | `>= 2.15.0`                  |
